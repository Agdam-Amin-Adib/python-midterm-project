# Expense Management System  

A Python-based application designed to manage personal or group expenses with a modern user interface and insightful visualizations.  

## Features
- Add, Edit, Delete, Search, and Sort expenses  
- Interactive Summary Dashboard  
- Visual reports with Pie Chart & Bar Chart  
- Date Range filtering for total calculations  
- User-friendly interface with modern design
